📘 How to flash ESP32 (step-by-step)
  1. Install Arduino IDE
  2. Add ESP32 board: File → Preferences → Add URL: https://espressif.github.io/arduino-esp32/package_esp32_index.json
  3. Install LoRa library: Tools → Manage Libraries → search "LoRa by Sandeep Mistry"
  4. Open inertia_lora.ino
  5. Select board: ESP32 Dev Module
  6. Select port: /dev/ttyUSB0 (or COM3)
  7. Click Upload →
