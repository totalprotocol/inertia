🔧Hardware Setup
  • 📥 Download firmware — inertia_lora.ino
  • 📱 BLE firmware — inertia_ble.ino
  • 🔌 Wiring: ESP32 + SX1276 (guide)
  • ⚡ Flash via Arduino IDE or Web Installer

📱 С телефона через Bluetooth (рекомендуется)

1 Скачайте Serial Bluetooth Terminal из Google Play → 📲 Скачать бесплатно
2 Загрузите BLE прошивку (inertia_ble.ino) на ESP32 через компьютер (один раз)
3 Включите Bluetooth на телефоне
4 Откройте Serial Bluetooth Terminal → Devices → "INERTIA Node"
5 ✅ Подключено! Теперь нажимайте "LORA CONNECT" ниже

💻 С компьютера через USB

1 Купите совместимую плату: LilyGO T-Beam ($35) или Heltec WiFi LoRa 32 ($25)
2 Подключите через USB к компьютеру
3 Нажмите "LORA CONNECT" ниже и выберите порт
4 Если порта нет — установите драйвер: ⬇️ CP210x Driver ⬇️ FTDI Driver
